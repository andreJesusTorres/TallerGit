Git

¿Qué es Git?

Git es un sistema de control de versiones que permite rastrear y gestionar cambios en archivos y proyectos de software. Permite a los equipos colaborar, mantener un historial de modificaciones y revertir cambios en el código de manera eficiente.

Front-End

¿Qué es Front-End?

El "front-end" es la parte de una aplicación o sitio web que los usuarios pueden ver y con la que interactúan, como el diseño y la interfaz en el navegador.

Back-End

¿Qué es Back-End

El "back-end" es la parte invisible de una aplicación o sitio web que maneja la lógica, la base de datos y la funcionalidad detrás de escena para hacer que todo funcione correctamente.
